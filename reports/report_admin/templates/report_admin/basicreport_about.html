Mission
To provide a fluid process for producing well researched, peer reviewed reports through a custom publishing workflow.

This platform provides an innovative report publishing application that leveraged the power and accuracy of the academic peer review process. Anyone can create an account and immediately author a report. The author must select a peer reviewer. The reviewer is responsible for suggesting changes and ultimately publishing the report.

A published report will be visible and accessible on the platform's home page. It can be retracted by the report's author or peer reviewer. The retracted report can then either be deleted or updated and re-published.

Features
ADMIN

Dashboard displays all reports connected to a user (author or reviewer)

SEARCH

Custom search with keyword(s), tags and/or date range. Makes use of Django/Postgres SearchVector

TAGS

Authors can attach keyword tags to a report. Other users can search via tags and view all reports grouped by the selected tag.

SUBSCRIPTION

Registered users can subscribe to tags and receive an email whenever a new report (with that tag) is published.
